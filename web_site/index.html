AmI Running

A beatiful Ambient Intelligent System 
designed to give you a better running experience

VISION

AmiRunning is a system designed for people that want a fully customized running/parkour experience on Skate Parks (eg. Piazza Valdo Fusi in Turin). 

Sensing the fatigue, blood pressure, heart rate, as long as the environment (such as weather, humidity..), the system is able to act on the park, modifying the obstacles (eg. the height or slope of platforms) and the path itself. 

In this way, the entire system is able to interact directly with the user, transforming the environment thanks to the infos collected during his experience. At the end of the run, a series of statistics (heart rate, temperature, humidity, lap time) will be showed.

MAIN STEPS

Sensing

The system takes data from environment, in particular humidity and weather, as well as userâ€™s conditions, mainly fatigue and hearth rate

Reasoning

The system, thanks to collected data, is able to decide what kind of changes is going to do on the system, in order to guarantee the best running/parkour experience.

Acting

The environment will change the obstacles height and will adapt to climatic changes, for example it will remove some obstacles if they aren't safe (after considering the weather). It will also modify the difficulty of the path according to the runner/traceur's conditions (considering physical conditions such as heart rate and fatigue).

Interacting

Before the training, the user can decide from 3 levels of difficulty, that will modify the obstacles. At the end of the run, a detailed report will be shown to the runner.

FEATURES

Sensitive

A smart collection of sensor will constantly monitorize both user and environment changes/conditions

Responsive

The acting on environment is made as fast as possible, to give a sense of "continuity" in the path to the runner

Adaptive

The system will continuously evolve and adapt depending on all the data coming from the environment

Intelligent

The system will smartly change in order to provide the best and safer path for the user

ARCHITECTURE

Central

All system is located in a central web server, where data are stored and managed

Distributed

The system is available from everywhere and everytime

Mobile

Everything is based on a very light and smoooth web application, to offer a good user experience

Our Raspberry Pi is the core of everything. The role of web server for our web application and circuit master is performed by it.

The role of Arduino Uno, with Adafruit Motorshield V2 upon it, is to control Steppers Motor installed. Acting as a Slave of entire circuit, his scope is to perform the requests made by Raspberry Pi

Thanks to user's FitBit Charge HR and other Sensors placed along the path, the system is able to adapt the obstacles and give a better running experience

With our stepper motors we are able to move obstacles and adapt them to you

HW COMPONENTS

  • Raspberry Pi
  • Arduino UNO
  • ZWave Sensors
  • FitBit Charge HR
  • Ultrasonic sensor
  • PIR sensor
  • Stepper motors
